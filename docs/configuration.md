# Configuration

Static local files live in:

```text
/etc/mnscloud/kamailio-webrtc/
```

Kamailio files live in:

```text
/etc/kamailio/
/etc/kamailio/mnscloud/
```

The sync process fetches:

```text
GET /api/v1/webrtc/edge/config
```

Expected request headers:

```text
Authorization: Bearer <node-token>
X-WebRTC-Node-UUID: <node-uuid>
```

Configuration must be generated by the MNSCloud API. Public examples must use
documentation-safe placeholders only.
